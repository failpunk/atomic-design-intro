StudyMode Frontend Today

  • Tightly Coupled
  • Hard to Test
  • Only Devs Can Modify UI
  • No Way to Version
  • Code Reuse Is Difficult
  • Deploys are Reliant on Essay Codebase

Make Our Own Bootstrap

  • Decouple from current codebase
  • Should slim down code size over time
  • Testing is easier
  • Regression testing possible
  • Easy to dev and deploy from CDN
  • Easy versioning of code and assets
  • Everything is a reusable module
  • Can be deployed indpendant of essays
  • Can be used on multiple sites
